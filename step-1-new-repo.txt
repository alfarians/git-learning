Cara buat repository baru:
  1. Buat repository baru dengan menekan ikon tanda "+" di sudut kanan atas dan pilih new repository.
  2. Beri nama untuk repository Anda [WAJIB]. Nama dengan spasi otomatis akan diganti dengan tanda dash "-", karena standar penamaan untuk alamat web harus menggunakan dash.
  3. Beri deskripsi (opsional) dab set siapa saja yang bisa mengakses dan melakukan commit: Pubic & Private.
  4. Cek "Initialize Readme" untuk membuat file Readme otomatis. File ini berisi deskripsi mengenai repository yang akan kita kerjakan, termasuk memasukkan deskripsi lisensi terkait hak intelektual yang kita terapkan di dalamnya untuk penggunaan, pengubahan, dan penjualan.
  5. "Add .gitignore" dan "Add licence" (opsional/sementara abaikan) -- advance using. 
  6. Terakhir tekan tombol "Create Repository". Repository baru siap digunakan.

Cara edit nama repository:
  1. Di bawah nama repository, pilih tab "Settings".
  2. Pada field "Repository Name", isikan nama baru dan tekan tombol "Rename".

Cara hapus repository:
  1. Di bawah nama repository, pilih tab "Settings".
  2. Geser ke bawah hingga pada kotak "Danger Zone". 
  3. Pilih tombol "Delete this repository".
  Catatan: Gunakan dengan hati-hati. Sekali terhapus, maka tidak bisa dikembalikan.
