Membuat branch cara #1:
	1. Membuat file baru dan/atau mengubah file yang sudah ada.
	2. Untuk pilihan tombol radio "Create new branch for this commit and start a pull request".
	3. Tekan tombol "Commit new file".

Membuat branch cara #2:
	1. Dari tampilan repo, pada tab "<> Code", tekan tombol "Branch: Master", isi pada field untuk mencari branch atau membuat baru.
	2. Bila belum ada branch yang dicari, akan tampil pilihan untuk membuat branch baru.
	3. Anda masuk ke tampilan untuk branch baru yang semuanya tampak sama dengan "Branch: Master", hanya nama pada "Branch: Nama Branch Baru".

Melihat daftar branch:
	1. Tekan sub tab "Branch" pada kolom "<> Code".

Menghapus branch:
	1. Tekan sub tab "Branch" pada kolom "<> Code".
	2. Tekan tombol "trash-bin" pada nama branch yang hendak dihapus.

Note:
	Additional reference: https://help.github.com/en/github/collaborating-with-issues-and-pull-requests/creating-and-deleting-branches-within-your-repository
